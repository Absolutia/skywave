# skywave
Chat program. Or its supposed to be. Doesn't really resemble one right now.
Currently for command line Linux/BSD only, but we have planned a DOS port as well as GUI versions for Windows & Android.
## libraries and liscensing
It relies on the functionality of some non-standard libraries (more will likely be joining that list in the future), and as such you will need to nab them to link skywave against.
We aren't going to supply them ourselves for obvious reasons, if I can I want to keep this BSD-liscensed.
If, uh, you notice a licensing problem let someone know ASAP and we'll get it fixed.
